# Space Shooter (Beginner Friendly)

Simple space shooter game written in Python using Pygame.

Features:
- Move left/right and shoot
- Enemies spawn from the top and move toward the player
- Score system and increasing difficulty
- Start and Game Over screens
- Easy to read modular code for students (10–15)

Installation
1. Create a Python 3.8+ virtual environment (recommended).
2. Install requirements:

```powershell
python -m pip install -r requirements.txt
```

Running
From the `space_shooter` folder run:

```powershell
python main.py
```

Controls
- Left arrow: move left
- Right arrow: move right
- Space: shoot
- Enter: start / restart
- Esc: quit

Assets
Put images in `assets/images` and sounds in `assets/sounds`.
If no assets are provided the game uses simple shapes and silent sounds so it still runs.
